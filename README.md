# Abandoned Web Catalog

Abandoned Web Catalog is a Rust console application that documents discontinued internet services, social networks and messaging platforms.

The application works completely offline and contains a small archive of well-known projects that are no longer available or have been discontinued.

## Features

- Browse archived services
- Search by project name
- Group by category
- Timeline view
- Summary statistics

## Categories

- Social Networks
- Messaging
- Web Services
